assaf-Pipes

Pipes service is a service that contain many conversion functions that can be used in any component.

Basic Usage

Step1 : Inject the service in your component's constructor

Step2 : Pipes service can be mainly used in assaf-datatable by providing the desired function to the pipe attribute

Hint : If you need to use date or decimal function you must to add DatePipe , DecimalPipe to AppModule providers Array.
